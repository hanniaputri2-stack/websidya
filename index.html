HTML

HTML atau HyperText Markup Langunge merupakan sebuah bahasa markah untuk membuat halaman web

CSS atau Cascading Stylesheet adalah bahasa stylesheet yang digunakan untuk mengatur tampilan, tata letak (layout), warna, font, dan dekorasi elemen HTML pada halaman web

Bootstrap adalah framework front-end open-source berbasis HTML, CSS, dan JavaScript yang paling populer untuk mengembangkan website responsif dan mobile-friendly dengan cepat.

PHP (PHP: Hypertext Preprocessor) adalah bahasa pemrograman scripting open-source yang berjalan di sisi server (server-side), dirancang khusus untuk pengembangan aplikasi web dinamis.

MySQL adalah sistem manajemen basis data relasional (Relational Database Management System/RDBMS) open-source yang populer, andal, dan cepat, digunakan untuk menyimpan serta mengelola data dalam tabel terstruktur. Menggunakan SQL (Structured Query Language), MySQL merupakan komponen utama dalam stack pengembangan web (seperti LAMP) dan banyak dipakai oleh aplikasi berskala besar seperti Facebook, Twitter, dan WordPress.

jQuery adalah pustaka (library) JavaScript yang ringan, cepat, dan kaya fitur, dirancang untuk menyederhanakan manipulasi Document Object Model (DOM), penanganan event, animasi, dan Ajax. Dengan semboyan "write less, do more", jQuery mempermudah pengembang web menulis kode JavaScript yang kompleks menjadi lebih ringkas dan kompatibel di berbagai browser.

AJAX (Asynchronous JavaScript and XML) adalah sekumpulan teknik pengembangan web yang memungkinkan aplikasi web memperbarui data di latar belakang (background) tanpa harus memuat ulang (reload) seluruh halaman. Ini membuat situs lebih cepat, interaktif, dan responsif karena hanya sebagian konten yang diubah.
